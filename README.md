## 📋 Présentation
**AgriExpert** est une solution intelligente conçue pour assister les agriculteurs dans l'identification des pathologies de plantation. L'application analyse les symptômes fournis par l'utilisateur et interroge une base de données d'expertise pour fournir un diagnostic précis et une solution curative.

## 🚀 Fonctionnalités
- **Analyse de symptômes** : Interface intuitive pour la sélection des signes cliniques observés.
- **Moteur de Diagnostic** : Logique de filtrage basée sur des données agronomiques structurées.
- **Recommandations** : Solutions pratiques (pollinisation manuelle, traitements, etc.).
- **Architecture de Données** : Persistance des diagnostics et des solutions en base de données.

## 🏗️ Architecture des Données
Le système repose sur une base de données relationnelle modélisée pour garantir l'intégrité des diagnostics :
- **Symptômes** : Table de référence des anomalies observables.
- **Diagnostics** : Table faisant la liaison entre symptômes et causes.
- **Solutions** : Base de connaissances des actions correctives.
